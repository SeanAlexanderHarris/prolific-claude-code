# Prolific Claude Code Marketplace Configuration

This directory contains the marketplace configuration for the Prolific Claude Code plugin.

## Files

- `marketplace.json` - Marketplace listing configuration

## Configuration Details

The marketplace configuration defines:
- Marketplace name and owner information
- Plugin listings and metadata
- Plugin sources and locations
- Tags for discoverability
- License information

## Usage

This configuration is used to publish the Prolific plugin to the Claude Code marketplace, making it discoverable and installable by Claude Code users.

## Publishing

To publish to the marketplace, submit this configuration according to the Claude Code marketplace publishing guidelines.
