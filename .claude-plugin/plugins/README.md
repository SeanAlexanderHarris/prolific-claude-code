# Prolific Claude Code Plugin Configuration

This directory contains the plugin configuration for the Prolific CLI integration with Claude Code.

## Files

- `plugin.json` - Plugin metadata and configuration

## Configuration Details

The plugin configuration includes:
- Plugin name and description
- Version information
- Author and repository details
- Keywords for searchability

## Installation

Users can install this plugin by:
1. Copying this directory to their Claude Code plugins directory
2. Installing via the Claude Code marketplace (once published)

## Requirements

- Prolific CLI installed and configured
- Valid Prolific API credentials
- Claude Code installed
